Quickstart

In order to quickly get started with Greymatter,

  1. Place the unzipped folder into your webserver location (probably something like ~/public_html)
  2. run the ./generate.sh shell script to properly create and chmod all of the required files
  3. Navigate to serverpath/DarkMatter/cgi-bin/gm.cgi to login to the default account (user: Alice pass: wonderland). On a local network, this would be something like: http://localhost/~john/DarkMatter/cgi-bin/gm.cgi
  4. Once logged in, click on the 'Configuration' button, check (and fix) the site URLs in the 'Path Configuration' table row, then save and run the initial diagnostic test. Once this finishes, you should be able to return to the main page and make your first post!
That's it! We hope you enjoy using Greymatter. To file an issue or bug, please navigate to https://github.com/JohnDDuncanIII/DarkMatter/issues
